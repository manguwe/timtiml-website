Hello World — C++ vs C

cout, cin, namespace · ~5 min
✓ Done ▼

C++ is built on top of C but adds object-oriented programming, the Standard Template Library, and modern features. The main difference you'll notice first: cout instead of printf.

C++
#include <iostream>   // C++ I/O library
using namespace std;    // lets us write cout instead of std::cout

int main() {
    // cout — C++ way to print (no format specifiers needed!)
    cout << "Hello, World! 🔧" << endl;
    cout << "Learning C++ with Anotida!" << "
";

    // cin — read user input
    string name;
    cout << "Enter your name: ";
    cin >> name;
    cout << "Hello, " << name << "!" << endl;

    return 0;
}
💡

C++ files use .cpp extension. You can use all C code inside C++ — C++ is a superset of C. The main upgrade: cout << is much easier than printf() with format specifiers.

Variables & Strings in C++

int, double, string, auto · ~5 min
✓ Done ▼
C++
#include <iostream>
#include <string>
using namespace std;

int main() {
    // Basic types — same as C
    int    age    = 20;
    double gpa    = 3.8;
    bool   active = true;  // C++ has real bool!
    char   grade  = 'A';

    // string — C++ string class (much easier than char arrays!)
    string name = "Anotida";
    string city = "Lusaka";

    cout << name + " from " + city << endl;
    cout << "Length: " << name.length() << endl;

    // auto — let C++ figure out the type
    auto x = 42;        // int
    auto pi = 3.14;     // double
    auto msg = "Hello"; // const char*

    // Constants
    const int MAX = 100;  // cannot change

    return 0;
}

03

Control Flow — If, Loops

if, for, while, range-for · ~6 min
✓ Done ▼
C++
#include <iostream>
using namespace std;

int main() {
    int score = 72;

    if (score >= 80)      cout << "Distinction!
";
    else if (score >= 60) cout << "Pass!
";
    else                   cout << "Fail!
";

    // Classic for loop
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
    cout << endl;

    // Range-based for (C++11) — like Python's for x in list
    int nums[] = {10, 20, 30, 40};
    for (int n : nums) {
        cout << n << " ";
    }
    cout << endl;

    // While loop
    int x = 1;
    while (x <= 100) x *= 2;
    cout << "First power of 2 > 100: " << x << endl;

    return 0;
}

04

Functions & References

function, & reference, default args · ~7 min
✓ Done ▼
C++
#include <iostream>
using namespace std;

// Function with default parameter
void greet(string name, string msg = "Welcome to TIMTIML!") {
    cout << "Hello " << name << "! " << msg << endl;
}

// Pass by reference — modifies original variable
void doubleIt(int& n) {   // & means reference
    n = n * 2;
}

// Function overloading
int    add(int a, int b)       { return a + b; }
double add(double a, double b) { return a + b; }

int main() {
    greet("Anotida");
    greet("Student", "Keep learning!");

    int x = 5;
    doubleIt(x);
    cout << "x = " << x << endl;  // 10

    cout << add(3, 4) << endl;      // 7
    cout << add(1.5, 2.5) << endl;  // 4.0
    return 0;
}

Classes & OOP in C++

class, constructor, access modifiers · ~9 min
✓ Done ▼
C++
#include <iostream>
using namespace std;

class Student {
private:                    // only accessible inside class
    string name;
    int    age;
    double gpa;

public:                     // accessible from outside
    // Constructor
    Student(string n, int a, double g)
        : name(n), age(a), gpa(g) {}   // initialiser list

    // Getters
    string getName() { return name; }
    double getGpa()  { return gpa; }

    // Method
    void introduce() {
        cout << "Hi! I'm " << name
             << " (age " << age
             << ", GPA: " << gpa << ")
";
    }
};

int main() {
    Student s1("Anotida", 20, 3.8);
    s1.introduce();
    cout << "Name: " << s1.getName() << endl;
    return 0;
}

06

STL — Vectors & Maps

vector, map, algorithm · ~7 min
✓ Done ▼

The Standard Template Library (STL) gives you powerful ready-made data structures. vector is like Python's list. map is like Python's dictionary.

C++
#include <iostream>
#include <vector>
#include <map>
#include <algorithm>
using namespace std;

int main() {
    // vector — dynamic array
    vector<int> scores = {90, 75, 88, 62, 95};
    scores.push_back(80);       // add element
    scores.pop_back();           // remove last
    cout << "Size: " << scores.size() << endl;

    // Sort and find max
    sort(scores.begin(), scores.end());
    cout << "Max: " << scores.back() << endl;

    // Range-for loop
    for (auto s : scores) cout << s << " ";
    cout << endl;

    // map — key-value pairs
    map<string, int> grades;
    grades["Anotida"] = 95;
    grades["Tinashe"] = 87;

    for (auto& pair : grades) {
        cout << pair.first << ": " << pair.second << endl;
    }
    return 0;
}

07

Pointers & Memory

*, new, delete, smart pointers · ~8 min
✓ Done ▼
C++
#include <iostream>
#include <memory>
using namespace std;

int main() {
    // Raw pointer (like C)
    int age = 20;
    int* ptr = &age;
    cout << "Value: "   << *ptr << endl;
    cout << "Address: " << ptr  << endl;

    // Dynamic memory (heap)
    int* heapNum = new int(42);  // allocate
    cout << *heapNum << endl;
    delete heapNum;                // ALWAYS free!
    heapNum = nullptr;            // best practice

    // Smart pointer (C++11) — auto-deletes! Preferred
    unique_ptr<int> smart = make_unique<int>(99);
    cout << *smart << endl;
    // No delete needed — auto-freed when out of scope

    return 0;
}
🏆

In modern C++ (C++11+), use smart pointers (unique_ptr, shared_ptr) instead of raw pointers. They automatically free memory — no more memory leaks! Ask Anotida for C++ inheritance and templates next.
