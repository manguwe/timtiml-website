01

Hello World — Your First C Program

printf, #include, main · ~5 min
✓ Done ▼

C is a compiled, low-level language. Unlike Python, you must compile C code into machine code before running it. Every program starts at main().

C
/* Include the standard input/output library */
#include <stdio.h>

/* main() — every C program starts here */
int main() {

    /* printf — print formatted text */
    printf("Hello, World!\n");
    printf("Learning C with Anotida at TIMTIML!\n");

    /* \n = newline character */

    return 0;  /* 0 means success */
}
⚡

To try C online without installing anything, go to onlinegdb.com — free online C compiler. Select language = C, paste code, click Run.

02

Variables & Data Types

int, float, char, double · ~6 min
✓ Done ▼

In C, you must declare every variable with its type before using it. C gives you very precise control over how much memory each variable uses.

C
#include <stdio.h>

int main() {

    int    age    = 20;          /* whole numbers */
    float  price  = 4.99f;       /* decimal (less precise) */
    double gpa    = 3.8;         /* decimal (more precise) */
    char   grade  = 'A';         /* single character */

    /* Format specifiers: %d=int, %f=float, %c=char, %s=string */
    printf("Age: %d\n", age);
    printf("Price: %.2f\n", price);  /* .2 = 2 decimal places */
    printf("GPA: %.1f\n", gpa);
    printf("Grade: %c\n", grade);

    /* String in C = array of chars */
    char name[] = "Anotida";
    printf("Name: %s\n", name);

    return 0;
}
💡

The format specifier (%d, %f, %s etc.) tells printf what type of data to print. Getting these wrong is a common C bug — always match the type!

03

Operators & Input

scanf, arithmetic, modulus · ~5 min
✓ Done ▼

C uses scanf() to read user input. The & (address-of) operator tells scanf where in memory to store the value — this is your first taste of pointers!

C
#include <stdio.h>

int main() {

    int a, b;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);  /* & = address of variable */

    /* Arithmetic operators */
    printf("Sum:      %d\n", a + b);
    printf("Diff:     %d\n", a - b);
    printf("Product:  %d\n", a * b);
    printf("Quotient: %d\n", a / b);
    printf("Modulus:  %d\n", a % b);  /* remainder */

    /* Increment / Decrement */
    a++;   /* a = a + 1 */
    b--;   /* b = b - 1 */

    return 0;
}

04

If / Else — Decision Making

if, else if, else · ~5 min
✓ Done ▼
C
#include <stdio.h>

int main() {

    int score;
    printf("Enter your score: ");
    scanf("%d", &score);

    if (score >= 80) {
        printf("Distinction!\n");
    } else if (score >= 60) {
        printf("Pass!\n");
    } else {
        printf("Fail. Keep studying!\n");
    }

    /* Logical operators: && (AND), || (OR), ! (NOT) */
    int age = 17, hasId = 1;  /* 1 = true in C */

    if (age >= 18 && hasId) {
        printf("Entry allowed\n");
    } else {
        printf("Entry denied\n");
    }

    return 0;
}
⚡

C has no bool type by default (before C99). Use 0 for false and any non-zero number (usually 1) for true. Or include #include <stdbool.h> for true/false.

05

Loops — for, while, do-while

iteration, break, continue · ~6 min
✓ Done ▼
C
#include <stdio.h>

int main() {

    /* for loop */
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    printf("\n");

    /* while loop */
    int n = 1;
    while (n <= 10) {
        printf("%d ", n);
        n += 2;   /* odd numbers: 1 3 5 7 9 */
    }
    printf("\n");

    /* Multiplication table */
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);

    for (int i = 1; i <= 10; i++) {
        printf("%d x %d = %d\n", num, i, num * i);
    }

    return 0;
}

06

Functions in C

return type, declaration, call · ~7 min
✓ Done ▼

C functions must be declared before they are used. You must specify the return type and parameter types explicitly.

C
#include <stdio.h>

/* Function declaration (prototype) — before main */
int add(int a, int b);
void greet(char name[]);
int factorial(int n);

int main() {
    printf("5 + 3 = %d\n", add(5, 3));
    greet("Anotida");
    printf("5! = %d\n", factorial(5));
    return 0;
}

/* Function definitions — after main */
int add(int a, int b) {
    return a + b;
}

void greet(char name[]) {
    printf("Hello, %s! 👋\n", name);
    /* void = returns nothing */
}

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);  /* recursion! */
}

07

Arrays & Pointers — The Heart of C

[], *, &, address · ~10 min
✓ Done ▼

Arrays store multiple values. Pointers store memory addresses. Pointers are what make C unique and powerful — understanding them unlocks deep programming knowledge.

C
#include <stdio.h>

int main() {

    /* ── ARRAYS ── */
    int scores[5] = {90, 85, 72, 95, 88};

    for (int i = 0; i < 5; i++) {
        printf("Score %d: %d\n", i + 1, scores[i]);
    }

    /* ── POINTERS ── */
    int age = 20;
    int *ptr = &age;  /* ptr stores the ADDRESS of age */

    printf("Value of age: %d\n", age);
    printf("Address of age: %p\n", ptr);
    printf("Value via pointer: %d\n", *ptr);  /* * = dereference */

    *ptr = 21;  /* change age through pointer */
    printf("Age is now: %d\n", age);  /* 21 */

    return 0;
}
🏆

Pointers are the hardest concept in C — but mastering them makes you a much better programmer in any language. Next steps: dynamic memory (malloc, free), structs, and file I/O. Ask Anotida for more!
