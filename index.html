Cambridge / ZIMSEC

O Level Computer Science

O Level Computer Science introduces you to the fundamentals of how computers work — from binary logic to writing your first programs. Whether you're studying Cambridge IGCSE or the Zambian/Zimbabwean curriculum, this course builds a rock-solid foundation for further studies and a career in tech.

  • Data representation — binary, hex, and character encoding
  • Computer hardware — CPU, memory, storage, I/O devices
  • Networking basics — LANs, WANs, the internet
  • Programming fundamentals — pseudocode and flowcharts
  • Algorithms — sorting, searching, and problem-solving
  • Cybersecurity — threats, malware, and safe practices
  • Database concepts — tables, queries, and SQL basics
  • Software development lifecycle
💬 Get Help with O Level CS
Advanced Level

A Level Computer Science

A Level Computer Science takes you deep into the theory and practice of computing. You'll explore complex algorithms, data structures, object-oriented programming, and computational thinking at a level that prepares you for university-level computer science and professional software development.

  • Advanced algorithms — complexity, Big O notation
  • Data structures — stacks, queues, trees, graphs, hash tables
  • Object-oriented programming — classes, inheritance, polymorphism
  • Functional programming concepts
  • Computer architecture — Von Neumann model, fetch-execute cycle
  • Operating systems — scheduling, memory management
  • Databases — relational theory, normalisation, SQL
  • Boolean algebra and logic circuits
  • Artificial intelligence introduction
