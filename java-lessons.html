Hello World — Your First Java Program

structure, main method · ~5 min
✓ Done ▼

Java is strongly typed and compiled. Every Java program lives inside a class, and execution starts at the main method. This structure is always the same.

Java
// Every Java file: filename must match the class name
// Save this as: HelloWorld.java

public class HelloWorld {

    // main() — Java starts here, ALWAYS
    public static void main(String[] args) {

        System.out.println("Hello, World! ☕");
        System.out.println("Learning Java with Anotida at TIMTIML 🚀");

        // print without newline
        System.out.print("No newline here. ");
        System.out.println("Back on same line.");
    }
}
⚡

Java is case-sensitive. String ≠ string. The file name MUST exactly match the class name. HelloWorld.java contains class HelloWorld.

Variables & Data Types

int, double, String, boolean · ~6 min
✓ Done ▼

Java is statically typed — you must declare the type of every variable before using it. This makes Java strict but reliable for large programs.

Java
public class Variables {
    public static void main(String[] args) {

        // Integer — whole numbers
        int age = 20;
        long bigNumber = 9876543210L;

        // Decimal numbers
        double price = 4.99;
        float gpa = 3.8f;

        // Text
        String name = "Anotida";
        char grade = 'A';   // single quotes for char

        // Boolean
        boolean isStudent = true;

        // Print with concatenation
        System.out.println("Name: " + name + ", Age: " + age);

        // Modern formatted string (Java 15+)
        System.out.printf("Price: %.2f%n", price);
    }
}
💡

Java has primitive types (int, double, boolean, char) and reference types (String, arrays, objects). Primitives start with lowercase, reference types with uppercase.

If / Else — Making Decisions

if, else if, switch · ~5 min
✓ Done ▼

Java's if/else works exactly like JavaScript/C. Use curly braces {} for the code block. Java also has a powerful switch statement.

Java
int score = 72;

if (score >= 80) {
    System.out.println("🏆 Distinction!");
} else if (score >= 60) {
    System.out.println("✅ Pass!");
} else {
    System.out.println("📚 Study harder!");
}

// Switch statement
String day = "Monday";

switch (day) {
    case "Monday":
        System.out.println("Start of the week 💪");
        break;
    case "Friday":
        System.out.println("Weekend soon! 🎉");
        break;
    default:
        System.out.println("Regular day.");
}

04

Loops — for, while, do-while

for, while, enhanced for · ~6 min
✓ Done ▼

Java has three loop types. The for loop is most common, the while loop runs when a condition is true, and the enhanced for loop (for-each) is perfect for arrays and lists.

Java
// for loop — classic
for (int i = 1; i <= 5; i++) {
    System.out.println("Count: " + i);
}

// while loop
int n = 1;
while (n <= 10) {
    System.out.print(n + " ");
    n++;
}

// Enhanced for (for-each) — iterates over arrays/lists
String[] languages = {"Java", "Python", "C++"};
for (String lang : languages) {
    System.out.println("I know: " + lang);
}

// do-while — runs at LEAST once
int x = 0;
do {
    System.out.println("Runs once even if false: " + x);
    x++;
} while (x < 0);

05

Methods — Reusable Code Blocks

static methods, return, parameters · ~7 min
✓ Done ▼

Methods in Java are like functions in other languages. Every method must declare its return type and parameter types. Use void when a method returns nothing.

Java
public class Methods {

    // void — returns nothing
    public static void greet(String name) {
        System.out.println("Hello, " + name + "! 👋");
    }

    // int — returns an integer
    public static int add(int a, int b) {
        return a + b;
    }

    // Method overloading — same name, different params
    public static double add(double a, double b) {
        return a + b;
    }

    public static void main(String[] args) {
        greet("Anotida");
        System.out.println(add(5, 3));       // 8
        System.out.println(add(2.5, 1.5));   // 4.0
    }
}
💡

Method overloading lets you define multiple methods with the same name but different parameter types. Java picks the right one based on what you pass in.

Classes & OOP — The Core of Java

class, object, constructor · ~10 min
✓ Done ▼

Java is an Object-Oriented Programming (OOP) language. Everything revolves around classes (blueprints) and objects (instances of those blueprints).

Java
// Class = blueprint
class Student {

    // Fields (attributes)
    String name;
    int age;
    double gpa;

    // Constructor — called when creating an object
    Student(String name, int age, double gpa) {
        this.name = name;
        this.age = age;
        this.gpa = gpa;
    }

    // Method
    void introduce() {
        System.out.println("Hi! I'm " + name +
            ", age " + age +
            ", GPA: " + gpa);
    }
}

public class Main {
    public static void main(String[] args) {

        // Create objects (instances) of Student
        Student s1 = new Student("Anotida", 20, 3.8);
        Student s2 = new Student("Tinashe", 19, 3.5);

        s1.introduce();   // Hi! I'm Anotida, age 20, GPA: 3.8
        s2.introduce();
    }
}
✅

The four pillars of OOP are: Encapsulation, Inheritance, Polymorphism, Abstraction. Once you master classes, study these — they're asked in every Java interview!

07

Arrays & ArrayLists

[], ArrayList, Collections · ~7 min
✓ Done ▼

Java has fixed-size arrays and the more flexible ArrayList. In real projects, ArrayList is preferred because it resizes automatically.

Java
import java.util.ArrayList;

public class Arrays {
    public static void main(String[] args) {

        // ── Fixed array ──
        int[] scores = {90, 85, 72, 95};
        System.out.println(scores[0]);          // 90
        System.out.println(scores.length);       // 4

        // Loop through array
        for (int s : scores) {
            System.out.print(s + " ");
        }

        // ── ArrayList — dynamic size ──
        ArrayList<String> langs = new ArrayList<>();
        langs.add("Java");
        langs.add("Python");
        langs.add("JavaScript");
        langs.remove("Python");

        System.out.println(langs.size());       // 2
        System.out.println(langs.contains("Java")); // true

        for (String lang : langs) {
            System.out.println(lang);
        }
    }
}
